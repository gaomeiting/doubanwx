# 微信小程序豆瓣电影

### 项目描述
> * 引导页
> * 榜单
> * 搜索
> * 我的
> * 电影列表
> * 电影详情
### 实现技术 wxml+wxss+json+js
### 技术要求
>* 1.自定义注册程序，封装api文件
>* 2.轮播图
>* 3.小程序路由控制
>* 4.代参数导航设置
>* 5.使用scroll-view的局部滚动
>* 6.多个异步请求同时加载 封装promise对象，promise.all([]).then().catch()
>* 7.页面传值；
>* 8.城市定位【百度地图】
>* 9.movie Class类的封装
>* 10.滚动加载onReachBottom
>* 11.定制键盘，confirm()
>* 12.template 提取
>* 13.画廊效果，大图预览,
>* 14.数据存储，收藏
>* 15.图片选择文件上传
